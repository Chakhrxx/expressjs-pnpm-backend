## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm dev

# production mode
$ pnpm start
```

## Test

```bash
# unit tests
$ pnpm test
```

## Features

- Robust routing
- Focus on high performance
- Super-high test coverage
- HTTP helpers (redirection, caching, etc)
- View system supporting 14+ template engines
- Content negotiation
- Executable for generating applications quickly
